# Power-BI-Task-Management-Dashboard
# 📊 Power BI Task Management Dashboard

## 📌 Overview

This project showcases a **Power BI dashboard** built to visualize daily task distribution using a **Pie Chart**.
It helps in understanding how time is allocated across different tasks in a simple and interactive way.

---

## 📁 Dataset Description

The dataset contains the following fields:

* **Task Name**
* **Hours Spent**
* **Percentage of Total Time**

### Example Tasks:

* Client Meeting
* Reply to Clients
* Orders Letter Preparation
* Preparing Quotation
* Group Meeting
* Sales Review
* Task Assignment
* Responding to Complaints
* Lunch Break
* Tea Break

---

## 📊 Dashboard Features

* Interactive **Pie Chart Visualization**
* Clear **Task Distribution Overview**
* Percentage-based insights for better decision-making
* Simple and clean UI design

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Microsoft Excel** (for data preparation)

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it in **Power BI Desktop**
3. Explore the dashboard and interact with the visuals
